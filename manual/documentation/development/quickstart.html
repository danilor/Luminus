El siguiente ejemplo muestra un breve tutorial de como crear módulos para Luminus, así como los grandes rasgos y definiciones que es necesario de utilizar.

Si se requiere ayuda para crear el módulo, existe un comando de artisan que crea el módulo y la estructura básica del módulo para la fácil utilización. El comando es:

php artisan Module create [NOMBRE]

Es recomendable ejecutar este comando con sudo si no se está seguro de que la carpeta tiene los permisos adecuados para la creación de los archivos necesarios.

Es importante señalar los siguientes cambios:

  • Para la nueva versión de Luminus se modifica el nombre de la clase de "DModule_Table" a "DModuleTable"
