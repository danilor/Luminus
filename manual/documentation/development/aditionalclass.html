El sistema permite el uso de clases adicionales para los módulos en uso. Basta con poner la clase deseada junto al módulo (siempre manteniendo el namespace de "Modules" como ancestro) y ésta será leída automáticamente por el sistema.

Es necesario el cuidar el nombre de la clase para que no tenga ningún problema con otras clases que puedan existir por parte de otros módulos.

Como recomendación, se puede utilizar un hijo de namespace para la nueva clase y así evitar conflictos con las ya existentes. Además, luego de añadir cada una de las clases nuevas es necesario ejecutar un composer update o un composer install para que la o las nuevas clases sean añadidas al class map.
