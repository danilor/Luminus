En ocaciones puede ser necesario que un módulo tenga presencia en todas las páginas de la aplicación, así como sus recursos. Esto por ejemplo para el caso de tener un módulo que presenta un Modal Window proveniente del menú superior.

Para este caso, la clase DModule permite establecer ciertos assets / recursos como globales. Estos recursos serán insertados justo después de los recursos princpiales de la página, y en orden de importancia de los módulos, como se establece en el sistema de orden de módulos.

Para realizar esto, se puede convocar a las siguientes funciones:

$this -> setGlobalCSS( ["css/bootstrap-dialog.css","css/global.css"] );
$this -> setGlobalJS( ["js/bootstrap-dialog.js","js/global.js"] );
Estas funciones deben ser llamadas en la sección Start() del módulo.

Con esto, el sistema incluirá estos recursos en los CSS y JS del sitio independientemente de la página donde se encuentre.

Estos assets están sujetos al cache del sitio. Si se realizan cambios en los assets, será necesario realizar una limpieza de caché para que estos sean cargados.
