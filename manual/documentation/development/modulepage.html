El Sistema de Luminus permite dentro de sus opciones, el establecimiento de una página principal (HOMEPAGE) para el módulo. Este es un valor opcional en la definición del módulo, y si es indicado, aparecerá en la lista de los módulos al momento de la instalación.

Para poder establecer una página principal del módulo se debe de establecer la variable de URL en el Archivo de Definición del Módulo.

Este detalle, lo que permite es añadir un ícono de mundo () junto al nombre del módulo con el enlace a la página principal del módulo.
