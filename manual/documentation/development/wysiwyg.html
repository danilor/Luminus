WYSIWIG es un acrónimo para "What you see is what you get" (Lo que vez es lo que obtienes). Luminus viene de manera predeterminada equipado con varias librerías WYSIWYG que pueden ser utilizadas a lo largo del sitio y por cualquiera de los módulos instalados.

SUMMERNOTE

Página web

Summernote es un plugin de JQUERY para convertir campos de texto o elementos como divs en visores WYSIWIG. Tiene la ventaja de ser muy liviano y además cuenta con una función para la subida de imágenes que no requiere trabajo BackEnd (es decir, no sube realmente las imágenes, sino que las transsforma a base64 y las inserta). Esta ventaja lo hace ideal para trabajos de blog, entradas, diarios y demás campos cuyos archivos se limitan solamente a la subida de imágenes.

De manera predeterminada, el sistema cuenta con una clase para summernote que, con solo aplicarla a las estructuras deseadas, debería de funcionar sin ningún problema

Ejemplo

<textarea class="summernote_wysiwyg" name="wysiwiyg_summernote_example" ></textarea>
