Los módulos tienen la capacidad de crear tareas programadas sin necesidad de añadie el comando a la lista de ejecuciones del sistema.

Para realizar esto, es necesario basarse en el método abstracto de setUpTasks que es solamente ejecutado una vez que el módulo es instalado.

public function setUpTasks()
{
   $task = new Task("update_currencies");
   /**
   * Esta tarea se va a ejecutar cada 30 minutos.
   */
   $task -> getSchedule() -> setUpFrequency( 0 , 0 , 30 );
   $this -> addTask( $task );
}

En el ejemplo anterior mostraro, la tarea va a ser ejecutada una vez cada 30 minutos. Los 0 anteriores al método de setUpFrequency indican días, horas y minutos respectivamente.

NOTA: la funcionalidad del sisetma de tareas automáticas está sujeta a cambios.

Una vez codificada la tarea, cuando el módulo es instalado, ésta va a ser almacenada en la tabla module_tasks con toda la información respectiva.
