El sistema posee además una serie de librerías orientadas a mejorar la experiencia de usuario respecto a los selectores. Aunque cada módulo tiene la libertad de usar cualquier librería (o no) que desee para mostrar estos campos, es recomendable utilizar las librerías suministradas por el sistema para mantener consistencia a través del sitio.

CHOSEN

Página web

CHOSEN es una librería basada en JQuery que permite modificar como son vistos los selectores de HTML (select) y brindar al usuario una mejor experiencia de usuario.

Para el sistema, basta con utilizar la clase: chosen-select en un select de html para poder habilitar la función.

<select class="chosen-select"> </select>
