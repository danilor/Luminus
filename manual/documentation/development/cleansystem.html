En oportunidades el sistema puede volverse lento al momento de manejar las peticiones de los clientes. Esto puede deberse en parte que por la naturaleza del mismo, Laravel crea constantemente archivos de caché y de soporte, que luego de un tiempo pueden llegar a llenar el disco duro del servidor.

Para elimiar estos archivos, se ha proporcionado un comando de artisan para poder limpiar el sistema de estos archivos. El comando es:

php artisan CleanSystem

El tiempo de duración de ejecución de este comando depende enteramente de la cantidad de archivos que estén siendo limpiados.
