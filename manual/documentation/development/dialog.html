GlobaSys tiene incluidas una serie de librerías JS que permiten el mostrar diálogos y mensajes rápidos desde Javascript sin necesidad de incluirlas en los módulos. Para ello, además de la inclusión de las librerías existen una serie de funciones y métodos predefinidos con los usos más comunes que se les pueden dar.

NOTY

Página web

NOTY es una librería basada en JQuery que permite mostrar notificaciones rápidas y simples para los usuarios. Es configurable y está incluida en el sistema para solamente ser utilizada. Además, GlobaSys cuenta con algunas funcionse predeterminadas para el uso de esta librería que deberían cubrir los aspectos más básicosd e estas notificaciones.

La función principal es:

function noty_element( text , layout , type ){
  var n = noty({
    text: text,
    layout: layout, // We are using the default layout
    type: type,
    timeout: 3000,
    theme:'metroui'
  });
}

La cual permite configurar el texto, el tipo de esquema y la estructura que se queire utilizar. Pero además, existen una serie de funciones básicas que lo que hacen es llamar a la anterior con valores predefinidos. Estas funciones son:

function noty_alert( text )
function noty_success( text )
function noty_error( text )
function noty_warning( text )
function noty_information( text)
function noty_notification( text )

Estas funciones preestablecen muchos de los valores que la librería NOTY permite y facilita a los creados de los módulos su uso.

Ejemplos

noty_alert
noty_success
noty_error
noty_warning
noty_information
noty_notification

Adicionalmente, si se envía sobre la URL los query string de save_message o error_message el sistema va a automáticamente generar un noty de success o error dependiendo de la situación.

service_details?id=1&save_message=Información+del+servicio+almacenada

BOOTSTRAP DIALOG

Página web

BOOTSTRAP es una librería basada en JQuery y Bootstrap que permite de manera más sencilla la creación de Modal Windows para su uso en la página web. Esta librería también está incluida en el sistema y puede ser utilizada en todos los módulos.

Por ejemplo:

BootstrapDialog.show({    message: 'Hi Apple! });

Adicionalmente, el sistema cuenta con una función que utiliza esta librería para facilidad. Es una función de confirmación de diálogo con redirección que se llama de la siguiente manera:

showDialogConfirmRedirect( title , text , url , block )

  • title el título de la ventana a mostrar
  • text el texto de la ventana a mostrar
  • url el url al cual redirigir en caso de confirmación
  • block true|false indica si queremos bloquear la página con una animación de carga

Otra función adicional creada es la siguiente:

showDialogMessage( title, text )

Al igual que la anterior, es una función que utiliza el Bootstrap Dialog para mostrar un mensaje simple. Este mensaje no posee botones y solo consta de in título y un texto.

Ejemplos

Recargar la página (showDialogConfirmRedirect)
Mensaje simple (showDialogMessage)
