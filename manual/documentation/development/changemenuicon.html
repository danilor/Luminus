Es posible, para los menú de los módulos, el modificar el ícono de cada uno de los items mostrados. De manera predeterminada, el sistema asigna el ícono de gears () a los nuevos menú.

Para poder realizar esta modificación, es necesario cambiar un poco la forma en que se asignan los menús al momento de la definición del módulo.

public function setUpMenuItems()
{
  $Menu = new DModuleMenu();
  $MenuItem = new DModuleMenuItem( "url" , "Título de la página" );
  $MenuItem -> setIcon( '<i class="fa fa-mouse-pointer"></i>' );
  $Menu -> addItemObject( $MenuItem );
  $this -> setMenu( $Menu );
}

Este ícono puede ser de FontAwesome (predeterminado en el sistema) o de cualquier otro conjunto de iconos que se desee (por ejemplo de IonIcons). Incluso, puede ser modificado de tal manera que acepte imágenes a gusto.

También se pueden modificar los íconos de la navegación superior del sitio. Si fuera el caso de que solo se desea usar los de la navegación superior, no se establece entonces la navegación normal del sitio.

$Navigation = new DModuleNavigation();
$Navigation -> setTitle( $this->getClassName() );
$Navigation -> setIcon( '<i class="fa fa-camera-retro"></i>' );
$MenuItem = new DModuleMenuItem( "show" , "Ver Galería y Citas" );
$MenuItem -> setIcon( '<i class="fa fa-picture-o"></i>' );
$MenuItem -> addAttribute("target","_blank");
$Navigation -> addItemObject( $MenuItem );
$this -> setNavigation( $Navigation );

Adicionalmente, se puede modificar también el elemento padre del menú. Utilizando el mismo método de "setIcon" pero para el menú en vez de para el MenuItem se puede lograr este cometido.

public function setUpMenuItems()
{
   $Menu = new DModuleMenu();
   $Menu -> setIcon( '<i class="fa fa-rebel"></i>' );
   $MenuItem = new DModuleMenuItem( "show" , "Contador" );
   $MenuItem -> setIcon( '<i class="fa fa-mouse-pointer"></i>' );
   $Menu -> addItemObject( $MenuItem );
   $this -> setMenu( $Menu );
}

NOTA: Si el módulo ya está instalado y solamente se necesita modificar el menú, es necesario ejecutar el comando de artisan para actualizar el menú.
